# 04｜验收清单：SEC 十公司单年财务指标计算

版本：v1.0  
用途：新对话验收 Codex 交付物。  
原则：先判断成果是否合格，再讨论下一步。

---

## 0. 验收 verdict

验收结论只能是三类：

```text
ACCEPT
ACCEPT WITH CAVEATS
REJECT
```

直接 `REJECT` 的情形：

- 缺少 `metrics_matrix.csv`。
- 缺少 `golden_results.csv`。
- 缺少 `exceptions_and_review_items.md`。
- 数值大量无证据链。
- 使用第三方数据或模型记忆补数。
- Golden assertion 失败但没有停机报告实际值。
- Scalability gate 失败，或新增同行业公司需要改 `scripts/sec_pipeline.py`。

---

## 0A. 验收顺序总览

### 第一层：十家公司功能验收

```bash
python scripts/10_run_golden_assertions.py
```

要求：

### 第二层：去公司特例验收

```bash
python tools/check_no_company_literals.py
```

要求：

```text
生产 Python 路径不得出现公司名、ticker、CIK、固定 accession、固定财年日期 literal
config/、tests/fixtures/、docs/、报告输出可以出现公司身份
outputs/scalability_audit.csv 记录 AST literal 扫描结果
```

期望：

```text
生产 extractor 不按 company name / CIK / ticker / accession / 固定财年日期分支；
行业知识以 registry、profile rule、concept/dimension probe、表头映射、量纲/恒等式断言存在。
```

### 第三层：第 11 家公司测试

新增一家同行业公司时，只允许改：

```text

```

不允许改：

```text

```

`repair_validation_results.csv` 中以下行为级 validation 必须 PASS：


## 6. Standard financial metrics 验收

重点检查：

- [ ] Revenue 命中 tag 已记录。
- [ ] YoY 使用同一或可解释候选链。
- [ ] Net income 使用三级候选链，并记录实际命中 tag。
- [ ] EBITDA proxy 明确为 operating income + D&A，不加回 impairment。
- [ ] FCF = OCF - capex。
- [ ] Debt-to-equity 未误用 DebtSecurities*。
- [ ] Interest coverage 在 EBIT/operating income <= 0 时标 NOT_MEANINGFUL。
- [ ] Current ratio 对 JPM 标 N_A_STRUCTURAL。
- [ ] Cash reserves 主口径为 cash and equivalents。

## 15. Verdict 规则

### ACCEPT

必须同时满足：

- 输出包完整。
- 核心 CSV 和报告存在。
- 所有数值有证据链。
- Enphase / Ford golden 通过。
- repair_validation_results 全 PASS。
- scalability_audit 无生产路径违规。
- stratified_audit 全 PASS。
- 没有第三方补数。
- coverage matrix 完整。
- 报告明确列出 caveats。

### ACCEPT WITH CAVEATS

适用于：

- 大部分指标完成；
- P0 repair validation 全 PASS；
- 分层抽样 audit 全 PASS；
- 少量文本/MD&A/DEF14A 抽取失败但已标 NOT_EXTRACTED；
- 部分 non-critical golden candidates 待人工确认；
- 有明确 exceptions 和下一步建议。

### REJECT

适用于：

- 缺核心文件；
- 大量指标无证据；
- 使用第三方数据或猜数；
- golden 失败后仍继续给出成功结论；
- repair_validation 有 FAIL 但报告不是 NO-GO；
- scalability gate 失败，或新增同行业公司要改 `scripts/sec_pipeline.py`；
- stratified_audit 有 FAIL 且未写入 exceptions；
- 错用 JPM current ratio、Salesforce ARR、Ford debt、Paramount YoY 等关键口径；
- 无法复现。
