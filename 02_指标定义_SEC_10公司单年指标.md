# 02｜指标定义：SEC 十公司单年财务、治理、风险与事件指标

版本：v1.0  
用途：锁定指标口径、公式、来源优先级、状态枚举和不可得处理。  
适用范围：10 家目标公司最近一个已申报财年。

---

## 0. 通用取数规则

### 0.1 报告期

- 绝大多数公司取 FY2025。
- Salesforce 财年末为 0131，取 FY2026。
- Macy's 财年末为 0201，使用最新 10-K 的 actual reportDate。
- Paramount Skydance / Paramount Global 需说明 successor/predecessor 口径；YoY 可标 `NOT_MEANINGFUL`。

### 0.2 companyfacts 取数算法

期间型指标：

```text
form 以 10-K 开头
end == 目标财年 period end
有 start
duration 约 300–400 天
若多条，取 filed 最新者
```

时点型指标：

```text
form 以 10-K 开头
end == 目标财年 period end
无 start 或 instant 型
若多条，取 filed 最新者
```

上年值：同一候选链、同一算法，end 取上一财年末。

所有命中的实际 taxonomy / concept / unit / accession / context 必须记录。

### 0.3 来源类型

```text
STD_XBRL       companyfacts 标准公司级事实
DIM_XBRL       accession instance 中的维度事实
CUSTOM_XBRL    accession instance 中公司自定义事实
MDA            MD&A / EX-99 表格或文本
DEF14A         DEF 14A / ecd XBRL / 委托书文本
8K_ITEM        8-K item
TEXT           10-K / 10-Q / 8-K 文本章节
DERIVED        由若干事实公式计算
NOT_AVAILABLE  SEC 未披露或未能找到
```

## A. Financial Institutions Metrics（仅 JPMorgan Chase）

### A01 Tier 1 capital ratio

定义：Tier 1 capital / risk-weighted assets。  
来源优先级：

1. accession XBRL/iXBRL instance 中 Basel / regulatory capital 维度事实；
2. MD&A / capital adequacy table；
3. 找不到则 `NOT_AVAILABLE_SEC` 或 `NOT_EXTRACTED`。

注意：entity-level companyfacts 可能没有当前可用值，不得用普通资产负债表估算。

### A02 CET1 ratio

定义：Common Equity Tier 1 capital / risk-weighted assets。  
来源：同 A01。  
必须注明 Basel 口径，例如 standardized / advanced approach。

### A03 Liquidity coverage ratio

定义：High Quality Liquid Assets / projected 30-day net cash outflows。  
来源：MD&A liquidity / capital resources / Basel disclosure。  
通常不是标准 companyfacts 指标。

### A04 Net interest margin

定义：net interest income / average interest-earning assets。  
来源优先级：

1. MD&A average balances table；
2. 若未抽到平均生息资产，可用 `net interest income / average total assets` 作为 `OK_APPROX`，必须标注 proxy。

### A05 ROA

定义：net income / average total assets。  
净利润：优先 `NetIncomeLoss`，否则按净利润通用候选链。  
平均总资产：`(Assets_t + Assets_t-1) / 2`。

### A06 ROE

定义：net income / average shareholders' equity。  
可以同时报告 common-shareholder 口径和 total equity 口径。

### A07 Net income trends

定义：本年净利润、上年净利润、同比变化。  
单年任务中的趋势定义为 YoY pair。

### A08 Fee income vs interest income

定义：noninterest income 与 net interest income 的相对占比。  
来源：银行专用 XBRL concepts 或 MD&A revenue composition。  
若命中概念不确定，状态用 `NEEDS_REVIEW`。

### A09 Non-performing loans / NPL ratio

定义：nonaccrual / non-performing loans divided by total loans。  
来源：credit risk table / XBRL concept inventory 正则 `/Nonaccrual|Nonperforming/`。  
贷款总额概念需从 JPM concept inventory 中确认，不得硬编码。

### A10 Loan loss reserves

定义：allowance for credit losses / loans。  
来源：allowance for credit losses 概念或 credit quality table。  
注意排除证券类 credit loss reserve。

### A11 AUM

定义：Assets under management。  
来源：MD&A / segment discussion。  
非 GAAP 标准事实，通常为文本/表格抽取。

### A12 Trading exposure

定义：VaR 或 trading exposure。  
来源：Item 7A / MD&A market risk table。  
无数值则 `NOT_EXTRACTED`。

### A13 Geographic exposure

定义：收入、资产或风险暴露按地区拆分。  
来源：实例中的 geographic dimensions 或 MD&A 分部表。

## B. Commercial Risk Metrics（9 家非金融公司）

### B01 Revenue

定义：财年收入。  
候选链：

```text
RevenueFromContractWithCustomerExcludingAssessedTax
-> Revenues
-> SalesRevenueNet
-> RevenueFromContractWithCustomerIncludingAssessedTax
```

命中 tag 必须记录。

### B02 Revenue YoY growth

定义：`(Revenue_t - Revenue_t-1) / Revenue_t-1`。  
若公司主体不可比，例如合并年 successor/predecessor，标 `NOT_MEANINGFUL`。

### B03 EBITDA margin

定义：`(Operating income + depreciation and amortization) / revenue`。  
本任务命名为 GAAP EBITDA proxy，不加回 impairment。  
D&A 候选链：

```text
DepreciationDepletionAndAmortization
-> DepreciationAmortizationAndAccretionNet
-> DepreciationAndAmortization
```

### B04 Net income

候选链：

```text
NetIncomeLoss
-> ProfitLoss
-> NetIncomeLossAvailableToCommonStockholdersBasic
```

若使用 `ProfitLoss`，需标注可能含 non-controlling interest。

### B05 Free cash flow

定义：`operating cash flow - capital expenditures`。  
OCF：`NetCashProvidedByUsedInOperatingActivities`。  
Capex 候选链：

```text
PaymentsToAcquirePropertyPlantAndEquipment
-> PaymentsToAcquireProductiveAssets
```

Ford 这类公司可能命中 `PaymentsToAcquireProductiveAssets`，必须允许。

### B06 Debt-to-equity

定义：`total debt / shareholders' equity`。  
总债务候选构成：

```text
LongTermDebt
或 LongTermDebtCurrent + LongTermDebtNoncurrent（二选一，避免重复）
+ ShortTermBorrowings
+ CommercialPaper
+ FinanceLeaseLiabilityCurrent
+ FinanceLeaseLiabilityNoncurrent
```

排除 `DebtSecurities*`，因为这通常是投资资产而非借款。  
Ford 需输出合并口径和工业口径（若 instance dimensions 支持）。

### B07 Interest coverage ratio

定义：`operating income / interest expense`。  
利息费用候选链：

```text
InterestExpense
-> InterestExpenseNonoperating
-> InterestExpenseDebt
```

若 operating income <= 0，状态为 `NOT_MEANINGFUL`。

### B08 Current ratio

定义：`current assets / current liabilities`。  
来源：`AssetsCurrent` / `LiabilitiesCurrent`。  
JPM 这种银行不适用，状态 `N_A_STRUCTURAL`。

### B09 Cash reserves

定义：cash and cash equivalents。  
主概念：`CashAndCashEquivalentsAtCarryingValue`。  
可另列 cash + short-term investments 备注，但不要混为主口径。

### B10 Occupancy rate（Marriott）

定义：occupied rooms / available rooms。  
来源：MD&A 或 8-K EX-99.1 operating statistics。  
优先口径：Marriott 10-K Lodging Statistics 表的 `Comparable Systemwide Properties / Worldwide` 绝对值。  
若只能抽到 regional breakdown 或同比变化，必须注明口径；若无法唯一选择 companywide 绝对值，标 `NOT_EXTRACTED`。

### B11 RevPAR（Marriott）

定义：revenue per available room。  
来源：MD&A 或 8-K EX-99.1。  
优先口径：Marriott 10-K Lodging Statistics 表的 `Comparable Systemwide Properties / Worldwide` 绝对值。  
必须保存带 `RevPAR` 表头和所选行的表格/句子原文片段。  
不得把 `RevPAR increased/decreased x%` 当作 USD 绝对值。

### B12 ARR / churn（Salesforce）

ARR 和 churn 未必在 SEC filing 中直接披露。  
可用 RPO / cRPO 作为替代指标，但必须明确：

```text
RPO != ARR
cRPO != ARR
```

churn 如果没有明确披露，状态 `NOT_AVAILABLE_SEC` 或 `TEXT_QUAL`。

### B13 Capacity utilization（Ford / Enphase）

定义：actual production / capacity。  
SEC 年报通常不披露可计算数值。  
若只存在定性描述，状态 `TEXT_QUAL`；若未披露，状态 `NOT_AVAILABLE_SEC`。

## C. Governance & Leadership（全部公司）

### C01 CEO / CFO changes

来源：8-K Item 5.02。  
输出事件清单：date、accession、item code、brief。

### C02 Board composition

来源：DEF 14A。  
本轮可输出董事人数、独立董事数、委员会信息；若无法可靠结构化，状态 `TEXT_QUAL` 或 `NEEDS_REVIEW`。

### C03 Executive compensation signals

来源：DEF 14A，优先 ecd XBRL facts。  
候选：CEO / PEO total compensation。  
若 ecd tags 不清楚，dump ecd inventory 并标 `NEEDS_REVIEW`。

### C04 Auditor changes

双路径：

1. 8-K Item 4.01；
2. 本年 vs 上年 10-K 的 `dei:AuditorName` instance fact。

比较规则：

- current/prior accession 和 period 必须写入证据链。
- 若目标 filing 是 10-K/A 且不含 auditor report，可回退同财年原始 10-K。
- 审计师名称比较需忽略标点和空白差异，例如 `KPMG LLP` 与 `KPMG, LLP` 视为同一名称。
- successor/predecessor CIK 不可硬拼；无法得到同 CIK prior 10-K 时标 `NEEDS_REVIEW`。

## D. Risk & Legal（全部公司）

### D01 Risk factors summary

来源：10-K Item 1A。  
输出标题级摘要或主题清单。  
状态 `TEXT_QUAL`。

### D02 Litigation disclosures

来源：Item 3 / legal proceedings / contingencies notes。  
若有 XBRL loss contingency accrual，记录概念和值；否则记录文本存在性。

### D03 Regulatory investigations

来源：Item 1A / Item 3 / notes。  
关键词包括 investigation、subpoena、inquiry、regulatory、SEC、DOJ、FTC、FDA 等。  
输出命中段落和判断。

### D04 Going concern statements

来源：审计意见、10-K 文本、going concern XBRL concepts。  
未披露 going concern doubt 通常是正常结果，应写“未披露持续经营疑虑”，不是 parse failure。

## E. Event Signals（全部公司，财年窗口内 8-K）

| ID | 信号 | 8-K item | 说明 |
|---|---|---|---|
| E01 | M&A announcements | 1.01 / 2.01 / 8.01 + 关键词 | 8.01 需正文关键词确认 |
| E02 | Bankruptcy filings | 1.03 | 预期多为零，零是正常结果 |
| E03 | Leadership departures | 5.02 | 与 C01 同源 |
| E04 | Financial restatements | 4.02 | 高风险信号 |
| E05 | Material agreements | 1.01 | 输出清单，不必全文抽合同条款 |

## F. 关键公司特殊规则

| 公司 | 特殊规则 |
|---|---|
| JPMorgan | FI track；current ratio 不适用；资本比率、NPL、allowance 需监管/信用风险表或维度事实。 |
| Ford | 债务需关注 automotive vs Ford Credit；B06 尽量给合并和工业双口径；亏损年 B07 可能 `NOT_MEANINGFUL`。 |
| Paramount | successor/predecessor 口径必须说明；YoY 可能不可比；事件扫描跨 CIK。 |
| Salesforce | fiscal year end 0131；ARR/churn 不得用 RPO 直接冒充。 |
| Macy's | fiscal year end 0201，实际 reportDate 以最新 10-K 为准。 |
| Marriott | B10/B11 优先走 10-K Lodging Statistics 的 Comparable Systemwide Properties / Worldwide 绝对值；不得用 RevPAR growth 冒充 USD。 |
| Enphase | 标准 XBRL 较干净，可作为标准指标 golden seed。 |
| Lumen | 减值、债务和持续经营/重组风险需重点标注。 |
| Pfizer | litigation、regulatory、R&D、product concentration 需要文本证据。 |
| Southwest | 航空运营指标不在核心清单，但可在 MD&A 中作为补充发现。 |
